Passo 1 — Exportar do Spotify

Use o Exportify para gerar o CSV

O Exportify é gratuito e não armazena seus dados.

  1. Acesse exportify.net e clique em Log in with Spotify
  2. Escolha a playlist que quiser
  3. Clique em Export → baixa um arquivo .csv
  4. Suba esse arquivo aqui embaixo ↓
Passo 2 — Como funciona a busca

Chordify via busca automática

Para cada música do seu CSV, este app abre o Chordify e extrai os acordes encontrados. Como o Chordify usa JavaScript dinâmico, o processo gera um link direto para você verificar as músicas marcadas como "não encontradas" manualmente.
